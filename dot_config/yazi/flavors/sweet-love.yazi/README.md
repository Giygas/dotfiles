## Preview

A warm, earthy dark theme with burnt orange, olive green, and muted gold tones.

## Installation

Copy the `sweet-love.yazi` directory into your Yazi flavors directory:

```sh
cp -r sweet-love.yazi ~/.config/yazi/flavors/
```

## Usage

Set the content of your `theme.toml` to enable it as your _dark_ flavor:

```toml
[flavor]
dark = "sweet-love"
```

Make sure your `theme.toml` doesn't contain anything other than `[flavor]`, unless you want to override certain styles of this flavor.

See the [Yazi flavor documentation](https://yazi-rs.github.io/docs/flavors/overview) for more details.
